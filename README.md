# Daily Todos

Simple todos app I created to track things I'd like to accomplish on a daily basis.

## Features

- [ ] Manage TODO list
  - [ ] Add Daily TODO
  - [ ] Edit Daily TODO
  - [ ] Delete Daily TODO
- [ ] Persist/load TODOS locally in browser
- [ ] Clear done TODOS on load of new day
- [ ] TODO Type - binary
- [ ] TODO Type - target
- [ ] TODO Type - multiple
