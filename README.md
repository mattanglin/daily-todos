# Daily Todos

Simple todos app I created to track things I'd like to accomplish on a daily basis.

## Features

- [x] Manage TODO list
  - [x] Add Daily TODO
  - [x] Edit Daily TODO
  - [x] Delete Daily TODO
  - [x] Manually clear TODO
- [x] Persist/load TODOS locally in browser
- [x] Clear done TODOS on load of new day
- [ ] TODO Type - binary
- [ ] TODO Type - target
- [ ] TODO Type - multiple
